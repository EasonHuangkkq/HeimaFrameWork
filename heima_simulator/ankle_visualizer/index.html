1 机构结构

踝关节是一个并联连杆机构。两个电机不是直接转动脚踝，而是通过两根刚性连杆间接驱动脚板。

// 脚板上的两个连接点（固定在脚板上，跟着脚板一起 pitch/roll） C = (-69, +35, 0) mm ← 脚板左侧，连着连杆E D = (-69, -35, 0) mm ← 脚板右侧，连着连杆F // 两个电机的轴心（固定在小腿上，不动） 电机E轴 = (-22, +34.5, 313.8) mm 电机F轴 = (-12, -34.5, 232.3) mm // 电机摇臂长度 摇臂长 = 65 mm // 连杆长度（刚性，永远不会变！） |CE| = 313.837 mm |DF| = 232.273 mm
🎯 类比理解
想象你用两根筷子去撬一块板子。筷子长度不会变（刚性）， 你只能转动筷子的根部（电机），筷子另一头连在板子上， 板子就会被撬到特定角度。
现在反过来：告诉你板子要倾斜多少度，算出两根筷子该怎么转——这就是求解器做的事。

2 问题：已知脚踝角度，求电机角度

pitch, roll → 脚板旋转 → C', D' 点移动 → ? → θ_e θ_f

脚板旋转后，C 和 D 点位置变了，但连杆长度不能变。 所以电机必须转到正确的角度，使得连杆刚好够得着新位置的 C' 和 D'。

// 脚板旋转后 C' 和 D' 的新位置 C' = Ry(pitch) × Rx(roll) × C D' = Ry(pitch) × Rx(roll) × D // 电机摇臂末端 E 和 F 的位置（随电机角度变化） E(θ_e) = Ry(θ_e) × (-65,0,0) + 电机E轴心偏移 F(θ_f) = Ry(θ_f) × (-65,0,0) + 电机F轴心偏移

3 核心约束：连杆长度恒定

连杆是刚性金属杆，长度不会变。这就是唯一的约束条件：

约束方程： f(θ_e) = |C' - E(θ_e)| - 313.837 = 0 g(θ_f) = |D' - F(θ_f)| - 232.273 = 0 // 翻译：找到 θ_e 使得 C'到E 的距离恰好等于连杆长度 // 翻译：找到 θ_f 使得 D'到F 的距离恰好等于连杆长度

4 牛顿迭代法求解

约束方程是非线性的（含 sin/cos），没有公式解。用牛顿迭代法数值求解：

牛顿法一句话： 从一个猜测值出发，每次修正一点，直到误差接近 0 迭代公式： θ_new = θ_old - f(θ_old) / f'(θ_old) 其中： f(θ) = 当前连杆长度 - 目标长度 ← 误差，要让它=0 f'(θ) = 误差变化的速度 ← 数值微分近似 f'(θ) ≈ ( f(θ+0.000001) - f(θ) ) / 0.000001
🎯 类比理解
想象你闭着眼调水温：
① 先随便拧一下（初始猜测 θ=0）
② 摸一下水温，发现太冷了（f(θ)=+18mm，连杆太长）
③ 往热的方向拧一些（θ_new = θ - 误差/变化率）
④ 再摸，还差一点（f(θ)=+0.3mm）
⑤ 微调一下（f(θ)=0.0001mm）
⑥ 完美！✅

5 逆向求解（读传感器时）

真机运行时方向反过来：电机编码器读到 θ_e, θ_f → 算出脚踝的 pitch, roll，然后喂给 RL 观测向量。

// 这次有 2 个未知数 → 用 2D 牛顿法 [pitch, roll]_new = [pitch, roll]_old - J⁻¹ × f(pitch, roll) // J 是 2×2 雅可比矩阵（每个约束对每个未知数的偏导数） J = ⎡∂f₁/∂pitch ∂f₁/∂roll⎤ ⎣∂f₂/∂pitch ∂f₂/∂roll⎦

原理一样，只不过从 1D 变成了 2D，需要 Jacobian 矩阵来确定两个方向的修正量。
